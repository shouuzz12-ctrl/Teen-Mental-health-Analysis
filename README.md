# Teen Mental Health Analysis Dashboard

## Project Overview
This project analyzes the relationship between social media usage and mental health among teenagers using SQL, Python, and Power BI.

## Tools Used
- SQL
- Python (Pandas, Matplotlib, Seaborn)
- Power BI

## Objectives
- Analyze screen time patterns
- Study anxiety and stress levels
- Compare social media platform usage
- Identify key mental health trends

## Key Insights
- Higher screen time was associated with higher anxiety scores.
- Mental health indicators varied across age groups.
- Social media usage patterns differed by platform.

## Files Included
- Dataset
- SQL Queries
- Python Analysis
- Power BI Dashboard
